# Machine Learning Engineer — (example lab)

## About the role

We're hiring ML engineers to build and scale training and inference systems for
large models. You'll work across the stack, from data pipelines to distributed
training to production serving.

## What you'll do

- Design and run large-scale model training jobs across many GPUs.
- Build data pipelines that feed training at scale.
- Optimize inference latency and cost for production model serving.
- Collaborate with researchers to turn prototypes into robust systems.

## What we're looking for

- Strong Python and software engineering fundamentals.
- Hands-on experience with PyTorch.
- Experience with distributed training (DDP, FSDP, or similar).
- Familiarity with Docker and containerized deployment.
- Comfort with GPUs, CUDA basics, and performance profiling.
- Bonus: experience serving models behind an API at scale.
